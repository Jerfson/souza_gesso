# Souza Gesso - Arte em Gesso Alto Padrão

Este é um projeto de landing page premium para a **Souza Gesso**, focada em serviços de gesso e design de interiores de alto padrão. O design foi inspirado na estética minimalista e fluida da Apple, utilizando tecnologias modernas para proporcionar uma experiência de usuário excepcional.

## ✨ Funcionalidades

- **Smooth Scroll**: Navegação suave utilizando a biblioteca [Lenis](https://github.com/darkroomengineering/lenis).
- **Parallax de Luxo**: Efeito de profundidade nas imagens controlado pelo scroll com [GSAP ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/).
- **Micro-interações**: Botões com efeitos de "beam" e cards com iluminação dinâmica (Glow Cards).
- **Design Responsivo**: Adaptado para todos os dispositivos usando [Tailwind CSS](https://tailwindcss.com/).
- **Animações de Entrada**: Revelação progressiva de elementos conforme o scroll (Aura v6 Engine).

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** (Custom Properties & Keyframes)
- **Tailwind CSS** (via CDN para prototipagem rápida)
- **JavaScript (ES6+)**
- **GSAP** (GreenSock Animation Platform)
- **Lenis** (Smooth Scroll)
- **Iconify** (Ícones em formato vetorial)
